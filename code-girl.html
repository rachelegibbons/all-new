Code Girl

Concept + Design

Code Girl follows high-school and college aged girls from across the globe create apps that catapult change in their community. Each team aims for their app to be selected for the Technovation Challenge finals. The Technovation Challenge aims to change the world by empowering girls worldwide to develop apps.

For the Portland community screening I created a promotional poster to promote the event to use as takeaways the night of the event.

The design mimics where diversity and technology intersect through using multiple spoken languages combined with programming languages and a neutral color palette. The hierarcical focus is on the women themselves.

All proceeds went to App Camp for Girls.
